# Leku Gerald's Personal Website

A modern, responsive personal website built with React and Tailwind CSS.

## Features
- Clean and modern design
- Responsive layout
- About section
- Portfolio showcase
- Contact form
- Social media links

## Technologies Used
- React
- Tailwind CSS
- Framer Motion
- React Icons

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
